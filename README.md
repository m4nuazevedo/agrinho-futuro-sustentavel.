# agrinho-futuro-sustentavel.
Projeto Agro Forte, Futuro Sustentavel
Este e um projeto de interface web desenvolvido no VS Code para demonstrar o equilibrio entre a potencia da producao agricola e a preservacao do meio ambiente. O site utiliza uma paleta de cores baseada em verde clarinho e azul, transmitindo frescor, tecnologia e sustentabilidade.

Funcionalidades
Design Responsivo: Interface limpa que se adapta a diferentes tamanhos de tela.

Elementos Interativos (Baloes): O site conta com baloes de informacao que, ao serem clicados, revelam detalhes especificos sobre Producao, Natureza e Futuro.

Painel de Informacoes Extras: Um botao interativo "SAIBA MAIS.." que expande um painel com dados adicionais sobre tecnologia no campo.

Visual Imersivo: Uso de imagem de fundo com efeito de rolagem (parallax) e camadas de transparencia para garantir a leitura e a estetica moderna.

Tecnologias Utilizadas
HTML5: Estrutura semantica do conteudo.

CSS3: Estilizacao com foco em Flexbox, transicoes suaves e filtros de cor (RGBA).

JavaScript: Logica para manipulacao de classes e exibicao dinamica de conteudos escondidos.

Como Executar o Projeto
Faca o download dos arquivos index.html, style.css e script.js.

Certifique-se de que todos os arquivos estao na mesma pasta.

Abra o arquivo index.html em qualquer navegador ou utilize a extensao Live Server do VS Code.
